# Authentication

## Prerequisite

`Nodejs:` For Running a js
`Postman:` Testing an API
`express:` Backend JS framework
`nodemon:` For constantly running backend
`mongoose:` Interacting with mongodb database
`dotenv:` Managing env file
`jsonwebtoken:` Used for authentication
`bcrypt:` For hashing the password
`nodemailer:` handling mail
`cors:` for managing connection between frontend and backend
